## Description
Build AI microservice for donor scoring and matching

## Features
- Donor reliability scoring (0-100)
- Response time prediction
- Optimal donor matching algorithm
- Allocation optimization

## Endpoints
- `POST /api/ai/score` - score donor
- `POST /api/ai/match` - find optimal donors
- `POST /api/ai/allocate` - optimize blood allocation

## Scoring Factors
- Distance to hospital
- Historical response time
- Donation streak
- Reliability score
- Blood group rarity
